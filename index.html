01 · ภาพรวมผลิตภัณฑ์

Personal AI Agent ที่องค์กรเป็นเจ้าของ 100%

Softnix PrivateClaw ให้ผู้ใช้งานทุกคนมี AI Agent ส่วนตัวที่ทำงานภายในองค์กรแบบ On-Premise เข้าถึงข้อมูลภายในอย่างปลอดภัย พร้อม Harness Agent อัจฉริยะที่ทำงานอัตโนมัติทั้งแบบตั้งเวลาและแบบ Workflow

100% On-Premise

ทำงานภายในองค์กรด้วย LLM Model ที่อยู่ภายในทั้งหมด ข้อมูลไม่ออกนอกองค์กร

Privacy & Security

รักษาความลับข้อมูลธุรกิจและข้อมูลส่วนบุคคล (PDPA) ให้อยู่ภายในองค์กร

Autonomous Work

Harness Agent ทำงานอัตโนมัติ ตั้งเวลาหรือทำงานแบบ Workflow แทนพนักงาน

Self Improvement

เรียนรู้จากการใช้งานของผู้ใช้ นำมาปรับวิธีการทำงานให้เร็วและฉลาดขึ้นต่อเนื่อง

Organizational Memory

เก็บทักษะและความรู้สำคัญไว้ในองค์กร ความรู้ไม่สูญหายเมื่อพนักงานลาออก

Enterprise Control Plane

ผู้ดูแลระบบควบคุม AI ได้ทั้งต้นทุน ความปลอดภัย และการกำกับดูแล

สิ่งที่พนักงานได้รับ

ผู้ช่วยที่เข้าใจบริบทงานของคุณ

  • สรุปข้อมูล & แจ้งเตือน สรุปเอกสาร/การประชุม และแจ้งเตือนเหตุการณ์สำคัญให้อัตโนมัติ
  • จัดการนัดหมาย ช่วยจัดตารางงานและนัดหมายผ่าน Calendar Connector
  • จดจำการสนทนา (Memory) จดจำบริบทการทำงานเดิม ต่อยอดงานได้ต่อเนื่อง

หน้าเริ่มต้นใช้งาน พร้อมคำแนะนำงานอัจฉริยะจาก Claw

02 · ความสามารถหลัก

Harness Agent & Skills — AI ที่ลงมือทำงานจริง

PrivateClaw ไม่ใช่แค่ผู้ช่วยตอบคำถาม แต่เป็น AI ที่ลงมือทำงานแทนได้จริง ผ่าน Harness Agent ที่วางแผนและลงมือทำงานเป็นขั้นตอน (Plan & Execution) แบบอัตโนมัติ และ Skill ที่ผู้ใช้สร้างเองเพื่อให้ AI ดำเนินการงานเฉพาะทางแทน

Scheduled Automation

ตั้งเวลาให้ AI ทำงานประจำอัตโนมัติ เช่น สรุปงานรายวัน แจ้งเตือน

Workflow Execution

ทำงานหลายขั้นตอนต่อเนื่องเป็น Workflow ลดงานซ้ำซ้อนของพนักงาน

Self Improvement

เรียนรู้จากการใช้งานจริง ปรับวิธีทำงานให้เร็วและแม่นยำขึ้นเรื่อย ๆ

Enterprise Knowledge

เข้าถึงคลังความรู้และข้อมูลองค์กรได้รวดเร็วภายใต้สิทธิ์ที่กำหนด

Skills — ให้ AI ดำเนินการแทน

ผู้ใช้สร้าง Skill เพื่อให้ AI ทำงานเฉพาะทางแทนได้หลากหลาย ช่วยเพิ่มประสิทธิภาพ ลดต้นทุน และสร้างความได้เปรียบเชิงแข่งขัน

สรุปการประชุม สร้างใบเสนอราคา สร้างรายงาน ตรวจสอบเอกสาร แปลงรูปแบบข้อมูล หาข่าว / Research สร้างรูปภาพ เขียนอีเมล และอื่น ๆ อีกมากมาย

Harness Agent วางแผนและลงมือทำงาน 19 ขั้นตอน — สร้างเอกสาร Word อัตโนมัติ

Connectors & Models

เชื่อมต่อแอปและโมเดลที่องค์กรใช้อยู่แล้ว

เชื่อมต่อกับเครื่องมือที่พนักงานใช้ทุกวัน และเลือกใช้ LLM ได้หลากหลาย — ทั้งโมเดลภายในองค์กรและผู้ให้บริการภายนอก ภายใต้การควบคุมขององค์กร

Connectors — GitHub, Notion, OneDrive, Gmail, Outlook / Microsoft 365, Calendar, Web Search และอื่น ๆ

My Models — รองรับ OpenAI, Claude, Gemini, Softnix GenAI, OpenRouter และ vLLM / Ollama

03 · Enterprise Control Plane & Governance

ควบคุม AI ทั้งองค์กรจากศูนย์กลาง

ผู้ดูแลระบบสร้าง User Claw (Personal AI Agent) ให้ผู้ใช้แต่ละคน กำหนดนโยบายการใช้งาน สิทธิ์การเข้าถึงข้อมูล และควบคุมความปลอดภัยได้จากศูนย์กลาง

Guardrails — masking/blocking ข้อมูลอ่อนไหว เช่น บัตรเครดิต อีเมล และข้อมูลส่วนบุคคล

Plan Ladder — กำหนดเพดานข้อความ/รูปภาพ/ความถี่ ต่อแผนการใช้งาน

Dashboard วิเคราะห์การใช้ Token รายผู้ใช้ / โมเดล / ผู้ให้บริการ

Users & Claw

สร้างและจัดการ AI Agent ส่วนตัวให้ผู้ใช้แต่ละคนในองค์กร

Plans & Quota

กำหนดโควต้าการใช้งานต่อวัน/เดือน/ปี และเพดานค่าใช้จ่าย

Guardrails (PDPA)

นโยบายความปลอดภัย กรองเนื้อหาไม่เหมาะสม/ความลับ/ข้อมูลส่วนบุคคล

Access Control

กำหนดสิทธิ์การเข้าถึงข้อมูลและ Connector ในแต่ละระดับ

LLM Providers

เพิ่มโมเดลของตัวเองได้ เช่น ผ่าน Token API จากผู้ให้บริการอื่น

Usage & Audit Logs

ติดตามการใช้ Token, กิจกรรม และบันทึกการตรวจสอบย้อนหลัง
